# 实验规范

每次正式实验必须写入独立运行目录，并记录：

- 实验ID和运行ID；
- Git提交哈希与工作区是否干净；
- 数据集路径、版本和SHA-256；
- 模型、Provider、Embedding、温度、topK、提示词版本和其他参数；
- 操作系统、Java/Node版本和关键服务版本；
- 开始/结束时间、原始输出、异常和人工评分表；
- 汇总脚本版本与最终指标。

原始结果只追加、不手工修改。图表与论文表格必须由汇总结果生成。失败运行保留并标记原因。

## 推荐实验规模

- 至少5个外贸英语主题。
- 约50个经过定位的教材片段。
- 约100道候选习题，最终规模可在预注册时根据成本调整。
- 若进行人工评价，先固定评分量表和评价者安排，再查看模型结果。
